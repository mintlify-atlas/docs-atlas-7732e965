<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get HERCULES SGI up and running in your institution</p>
  
  <Steps>
    <Step title="Review System Requirements">
      Check the prerequisites for deploying HERCULES SGI, including Java 11+, PostgreSQL, and Keycloak authentication server.
      
      See the [Installation Guide](/deployment/installation) for detailed requirements.
    </Step>
    
    <Step title="Deploy Backend Services">
      Deploy the microservices architecture using Docker containers or build from source with Maven.
      
      ```bash
      # Clone the repository
      git clone https://github.com/herculesproject/sgi.git
      cd sgi
      
      # Build all services
      mvn clean install
      ```
      
      Each service runs independently and communicates via REST APIs.
    </Step>
    
    <Step title="Configure Authentication">
      Set up Keycloak with the SGI realm configuration to enable user authentication and role-based access control.
      
      Import the provided realm configuration from `sgi-auth/realm/sgi-realm.json`.
      
      Learn more in the [Authentication Guide](/deployment/authentication).
    </Step>
    
    <Step title="Access the Web Interface">
      Navigate to the SGI web application and log in with your configured credentials to start managing research activities.
      
      Explore the [User Guides](/guides/researcher) for role-specific workflows.
    </Step>
  </Steps>
</div>
